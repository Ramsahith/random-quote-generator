# 💬 Random Quote Generator

This is not just a quote generator – it’s a powerful mini web app that speaks, copies, and even tweets your quotes!

---

## 🔥 Features

- 🔄 **Random Quotes** – Fetched from an external API.
- 🗣️ **Text-to-Speech** – Speaks the quote aloud using SpeechSynthesis.
- 📋 **Copy to Clipboard** – Copies the current quote with one click.
- 🐦 **Tweet Quote** – Loads the quote into a new tweet via Twitter Intent.
- 🎨 **Responsive UI** – Neat layout with clean styles.

---

## 🧑‍💻 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Fetch API, Web Speech API, Clipboard API)  
- Twitter Web Intent  

---



## 🚀 How to Run

1. Download or clone the repository.  
2. Open `index.html` in your browser.  
3. Enjoy the quotes, copy, listen, or tweet them instantly!
